A web 2.0 way of enhancing users interaction on submitting your posts to Social services.

== Description ==

**Hey Social** is the coolest way to interact with the visitors of your blog, so that they can submit your posts and pages to the most famous Social services, such as *Digg*, *Twitter*, *De.licio.us* and so on. 

This plugins makes heavy use of Javascript effects to impress visitors with the latest tecnique of *Web 2.0*.

This plugin is based on the great work [HeySilver](http://www.anieto2k.com/heysilver/index_en.html), a JS class which uses mootools APIs to affascinate people with an inline navigation menù a là **QuickSilver**.

== Installation ==

1. Extract the Archive
2. Copy the whole *hey-social* folder to your /wp-content/plugins folder
3. Go to http://yoursite.com/wp-admin/plugins and activate *hey social*

== Frequently Asked Questions ==

It is stable as it works and does not have bugs, however it is still in graphic and functional developing
